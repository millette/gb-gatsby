# README
> Gitbook clone based on gatsby.

```sh
# Download Gatsby generator
git clone https://github.com/millette/gb-gatsby.git

# Download content (example)
git clone https://github.com/jhroy/syllabus-EDM5240-H2019.git content

# Download node dependencies
npm install

# Edit gatsby-config.js to suit your needs

# Demo on http://localhost:8001/
npm start

# Production build in public/
npm run build
```
